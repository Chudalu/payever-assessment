
## Description

Assessment code for Payever backend engineer role.
CHECK OUT Swagger API Docs at - http://localhost:3500/docs, This would help to see all APIs and all required bodies and responses.
VISIT app.config.ts to update database URL for mongoDB

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
